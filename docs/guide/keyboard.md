# Keyboard Shortcuts

## Chat

| Shortcut | Action |
|---|---|
| `Enter` | Send message |
| `Shift+Enter` | New line in input |
| `Escape` | Stop generation |
| `Ctrl+Shift+C` | Copy last response |

## Navigation

| Shortcut | Action |
|---|---|
| `Ctrl+N` | New conversation |
| `Ctrl+K` | Search conversations |
| `Ctrl+M` | Quick model switcher |
| `Ctrl+↑` / `Ctrl+↓` | Navigate to previous / next conversation |
| `Ctrl+/` | Toggle sidebar |
| `Ctrl+H` | Open Hosts settings |
| `Ctrl+,` | Open Settings |

## View

| Shortcut | Action |
|---|---|
| `Ctrl+Shift+M` | Toggle Compact / TWM Mode |

## Drag & Drop

Drop files directly onto the chat input area:

| File type | Result |
|---|---|
| Images (`.png`, `.jpg`, `.gif`, `.webp`, …) | Attached to message |
| Text files (`.txt`, `.md`, `.py`, `.ts`, `.json`, …) | Linked as file context |

Multiple files can be dropped at once. Folders must be linked via the attach menu.

## Paste (Ctrl+V)

Paste works anywhere on the chat page — focus does not need to be in the input:

| Clipboard content | Result |
|---|---|
| Image data (e.g. screenshot) | Attached to message |
| Files copied from file manager | Images attached; text files linked as context |
| Plain text | Inserted into input field normally |

Multiple files pasted at once are all processed.
